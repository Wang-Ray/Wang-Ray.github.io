### -i

用来编辑文件。

查找替换

```shell
# 第一个
$ sed -i 's/原字符串/新字符串/' file
# 全局
$ sed -i 's/原字符串/新字符串/g' file
```

特定行查找替换

```shell
$ sed -i '/匹配字符串/s/原字符串/新字符串/' file
```

特定行行前插入行

```shell
$ sed -i '/匹配字符串/i 新行字符串' file
```

特定行行后插入行

```shell
$ sed -i '/匹配字符串/a 新行字符串' file
```

删除特定行

```shell
$ sed -i '/匹配字符串/d' file
```



### -E

abc.txt

```
c       e
```

测试

```shell
# 不成功，由于不支持+
$ sed -i '/c\s+e/a abc' abc.txt 
# 成功，由于支持+
$ sed -E -i '/c\s+e/a abc' abc.txt 
# 成功，由于支持*
$ sed -i '/c\s*e/a abc' abc.txt
```
