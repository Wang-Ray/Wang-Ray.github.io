查询软件

```shell
# 显示包的版本
$ dpkg -l package
# 列出与该包先关联的文件
$ dpkg -L package
# 查找包的详细信息
$ dpkg -s package
# 列出deb包的内容
$ dpkg -c package.deb
```

安装软件

```shell
$ dpkg -i *.deb
```

卸载软件

```shell
# 保留配置
$ dpkg -r *.deb
# 不保留配置
$ dpkg -P *.deb
```
