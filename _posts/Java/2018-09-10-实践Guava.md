[Guava](https://github.com/google/guava) Google core libraries for Java



```xml
<dependency>
  <groupId>com.google.guava</groupId>
  <artifactId>guava</artifactId>
  <version>27.1-jre</version>
  <type>bundle</type>
</dependency>
```



```xml
<dependency>
  <groupId>com.google.guava</groupId>
  <artifactId>guava</artifactId>
  <version>27.1-android</version>
  <type>bundle</type>
</dependency>
```
