### 计算属性

基于computed会缓存结果，而基于method每次都会调用

```html
<p>$: {{ rmb/exchange }}</p>
<p>$: {{ getDollar() }}</p>
<p>$: {{ dollar }}</p>

<script>
new Vue({
      el: '#div1',
      data: {
          exchange:6.4,
          rmb:0
      },
      methods:{
          getDollar:function() {
              return this.rmb / this.exchange;
          }
      }
      computed:{
          dollar:function() {
              return this.rmb / this.exchange;
          }
      }
    })
</script>
```
