[PCSX2](https://pcsx2.net/)/[PCSX2 Emulator](https://pcsx2-emulator.com)  is a free and open-source PlayStation 2 (PS2) emulator. Its purpose is to emulate the PS2's hardware, using a combination of MIPS CPU Interpreters, Recompilers, and a Virtual Machine that manages hardware states and system memory.

The project has been running for almost 20 years. Past versions could only run a few game demos, but newer versions can run most games at full speed, including titles like Final Fantasy X and Devil May Cry 3.

A significant majority of the PS2 library is considered playable. For more info on compatibility, see [here](https://pcsx2.net/compat).

PCSX2 allows you to play PS2 games on your PC with added features like:

- Custom resolutions and upscaling
- Virtual and shareable memory cards
- Save-states
- Patching system
- Internal recorder for lossless quality at full speed
