[RetroArxh](https://www.retroarch.com/) is a frontend for emulators, [ game](https://www.retroarch.com/#) engines and media players.

Among other things, it enables you to run classic [ games](https://www.retroarch.com/#) on a wide range of computers and consoles through its slick graphical interface. Settings are also unified so configuration is done once and for all.

In addition to this, you are able to run original game discs (CDs) from RetroArch.

RetroArch/Libretro is an open-source project and has been around since 2012. It has since served as the backend technology to tons of (unaffiliated) platforms and programs around the world.
