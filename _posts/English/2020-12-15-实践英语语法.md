## 词性

### 📊 **英语九大词性分类及功能概览**

|  **词性**  | **缩写** |            **主要功能**             |        **典型例词**        |           **例句**            |
| :--------: | :------: | :---------------------------------: | :------------------------: | :---------------------------: |
|  **名词**  |    n.    |    表示人、事物、地点或抽象概念     | book, China, love, teacher | The **book** is interesting.  |
|  **代词**  |  pron.   |         代替名词，避免重复          |  I, she, they, this, mine  |     **She** is a doctor.      |
|  **动词**  |    v.    |        表示动作、状态或存在         |   run, be, study, become   |    They **study** English.    |
| **形容词** |   adj.   |      修饰名词或代词，描述特征       |   happy, tall, beautiful   | It is a **beautiful** flower. |
|  **副词**  |   adv.   |  修饰动词、形容词、其他副词或句子   |   quickly, very, always    |     He runs **quickly**.      |
|  **介词**  |  prep.   | 表示名词与其他词的关系（时间/地点） |      in, on, at, from      | The cat is **on** the table.  |
|  **连词**  |  conj.   |        连接单词、短语或句子         |   and, but, because, or    |  I like tea **and** coffee.   |
| **感叹词** | interj.  |         表达强烈情感或反应          |       oh, wow, alas        |   **Wow!** That's amazing!    |
|  **冠词**  |   art.   |        限定名词的特指或泛指         |         a, an, the         |    **The** sun is bright.     |

------

### 📌 **详细说明与注意事项**

1. **名词（Nouns）**

   - 功能：作主语、宾语、表语等（如：

     ```
     Dogs bark.中“dogs”为主语）。
     ```

   - 分类：

     - 专有名词：首字母大写（如China）。
     - 普通名词：分为可数（cat → cats）和不可数（water需用量词：a cup of water）。

2. **代词（Pronouns）**

   - 细分类型

     ：人称代词（I/me）、物主代词（my/mine）、指示代词（this/those）、疑问代词（who/what）等。

   - 注意：主格作主语（He runs），宾格作宾语（She helps him）。

3. **动词（Verbs）**

   - 分类：
     - 实义动词：及物（需宾语，如read books）和不及物（如birds fly）。
     - 助动词：辅助构成时态（is reading）或被动语态（was built）。
     - 情态动词：表能力或义务（can/must+动词原形）。

4. **形容词（Adjectives）**

   - 位置：定语（a red car）或表语（The car is red）。
   - 比较级：单音节词加-er/-est（tall → taller），多音节用more/most（beautiful → more beautiful）。

5. **副词（Adverbs）**

   - 修饰对象：
     - 动词（speak loudly）、形容词（very good）、句子（Fortunately, he won.）。
   - 类型：时间（now）、地点（here）、方式（carefully）等。

6. **介词（Prepositions）**

   - 核心作用：引出时间（at 7:00）、地点（in the room）、方式（by bus）等关系。
   - 固定搭配：需记忆（listen to,depend on）。

7. **连词（Conjunctions）**

   - 分类：
     - 并列连词：连接对等成分（and/but/or）。
     - 从属连词：引导从句（because/if）。

8. **感叹词（Interjections）**

   - 特点：独立于句子结构，常加感叹号（Oh! I forgot.）。

9. **冠词（Articles）**

   - 不定冠词：a/an表泛指（a book）。
   - 定冠词：the表特指（the book you bought）。

------

### 🔍 **其他补充词性**

- 数词（Numerals）：表示数量或顺序（one, first），用于时间、年龄等表达（five o'clock）。
- 量词（Quantifiers）：与不可数名词连用（a piece of paper），但现代语法常将其归入形容词或限定词。
- 疑问词（Interrogatives）：如when/why，部分归类为副词或代词。

### 名词（noun，n.）

#### 📌 一、**按意义分类** 

1. **专有名词（Proper Nouns）**
   - **定义**：表示特定的人、地点、机构、语言等独一无二的名称，首字母大写。
   - 示例：
     - `China`（中国）、`the Great Wall`（长城）
     - `English`（英语）、`United Nations`（联合国）
2. **普通名词（Common Nouns）**
   - 子类：
     - 个体名词（Individual Nouns）：表示同类事物中的个体。
       - 例：`teacher`（老师）、`computer`（电脑）
     - 集体名词（Collective Nouns）：表示由个体组成的集合体。
       - 例：`family`（家庭）、`class`（班级）
       - 用法注意：
         - 强调整体时作单数（*The class is large.*）
         - 强调成员时作复数（*The class are discussing.*）
     - 物质名词（Material Nouns）：表示无法分割的实物材料。
       - 例：`water`（水）、`paper`（纸）
     - 抽象名词（Abstract Nouns）：表示性质、情感、概念等抽象事物。
       - 例：`courage`（勇气）、`time`（时间）

------

#### 🔢 二、**按可数性分类** 

1. **可数名词（Countable Nouns）**

   - **定义**：可直接用数目计算，有单复数形式。

   - 规则复数变化：

     |        词尾规则        |           变化方式            |               示例                |
     | :--------------------: | :---------------------------: | :-------------------------------: |
     |        一般情况        |            加 `-s`            |           book → books            |
     | 以 `s, x, ch, sh` 结尾 |           加 `-es`            |            box → boxes            |
     |   以“辅音字母+y”结尾   |    变 `y` 为 `i` 加 `-es`     |           city → cities           |
     |     以 `f/fe` 结尾     |   变 `f/fe` 为 `v` 加 `-es`   |          knife → knives           |
     |      以 `o` 结尾       | 有生命加 `-es`，无生命加 `-s` | potato → potatoes, photo → photos |

   - 不规则复数变化：

     - `man → men`, `child → children`, `sheep → sheep`（单复数同形）

2. **不可数名词（Uncountable Nouns）**

   - **定义**：不能用数目直接计算，无复数形式。
   - 量化表达：
     - 单位词 + `of`：`a cup of tea`（一杯茶）、`two bottles of water`（两瓶水）
     - 概数词：`much information`（大量信息）、`a little milk`（一点牛奶）
   - **常见类型**：物质名词（`bread`）、抽象名词（`knowledge`）、部分集体名词（`furniture`）

3. **兼类名词（双重属性）**

   - 可数与不可义意义不同：

     |      可数意义       |    不可数意义     |
     | :-----------------: | :---------------: |
     |  `a chicken`（鸡）  | `chicken`（鸡肉） |
     |  `a room`（房间）   |  `room`（空间）   |
     | `a glass`（玻璃杯） |  `glass`（玻璃）  |

------

#### 🧩 三、**按形式分类** 

1. **简单名词（Simple Nouns）**
   - 单一词汇构成：`father`（父亲）、`signal`（信号）
2. **复合名词（Compound Nouns）**
   - 由两个及以上词组合：
     - 连写：`housewife`（主妇）
     - 加连字符：`brother-in-law`（姐夫）
     - 分开写：`traffic light`（交通灯）
   - 复数规则：
     - 有主体名词时，主体名词变复数（`apple tree → apple trees`）
     - 无主体名词时，末词变复数（`grown-up → grown-ups`）

------

#### 📚 四、**名词的句法功能** 

名词在句中可充当多种成分：

- **主语**：`The students finished their homework.`
- **宾语**：`She bought a new car.`
- **表语**：`He is a doctor.`
- **定语**：`a school bus`（校车）
- **同位语**：`Mr. Lee, our principal, spoke first.`

------

#### ⚠️ 五、**使用注意事项**

1. **主谓一致**：
   - 可数名词复数 + 复数动词（*Books are useful.*）
   - 不可数名词 + 单数动词（*Water is essential.*）
   - “单位词 + 不可数名词”按单位词定单复数（*Two cups of tea are on the table.*）
2. **限定词搭配**：
   - 可数名词单数：`a/an, each, every`（*a book*）
   - 可数名词复数：`many, few, several`（*many books*）
   - 不可数名词：`much, little, a large amount of`（*much water*）
3. **所有格形式**：
   - `-’s` 所有格：表有生命事物的所属（*the teacher’s desk*）
   - `of` 所有格：表无生命事物的所属（*the title of the book*）
   - 双重所有格：表部分概念（*a friend of Tom’s*）

------

#### 💎 **总结：名词分类的意义**

掌握名词分类可避免常见错误：

1. 可数性混淆：如误用 a advice（应为 a piece of advice）；

2. 主谓不一致：如The news are（应为is）；

3. 所有格误用：如 the book of Tom（口语中更常用Tom’s book）。

   结合例句和规则练习，能显著提升语言准确性 🌟。

### 动词（verb，v.）

#### 🧱 一、按语法功能分类

1. **实义动词（Notional Verbs）**

   - **定义**：表示具体动作或状态，可独立充当谓语。

   - 子类：

     - 及物动词（Transitive Verbs, vt.）：需接宾语才完整。

       例：eat（I eat an apple.）

     - 不及物动词（Intransitive Verbs, vi.）：无需宾语。

       例：sleep（She sleeps well.）

     - 双宾语动词（Ditransitive Verbs）：需两个宾语（直接+间接）。

       例：give（He gave her a book.）

     - 宾补动词：需宾语+补语。

       例：elect（They elected him president.）

2. **系动词（Linking Verbs）**

   - **定义**：连接主语与表语，描述状态或特征。

   - **常见类型**：

     |    类型    |           例词           |          例句           |
     | :--------: | :----------------------: | :---------------------: |
     | 状态系动词 |           `be`           |    He is a teacher.     |
     | 感官系动词 | `feel`, `smell`, `sound` |  The cloth feels soft.  |
     | 变化系动词 | `become`, `grow`, `turn` |  She became a doctor.   |
     | 持续系动词 |     `keep`, `remain`     |       Keep quiet!       |
     | 表像系动词 |     `seem`, `appear`     |     He seems tired.     |
     | 终止系动词 |   `prove`, `turn out`    | The rumor proved false. |

3. **助动词（Auxiliary Verbs）**

   - **定义**：辅助实义动词构成时态、语态、疑问或否定。
   - 功能：
     - `be` → 构成进行时/被动语态（He is singing.）
     - `have` → 构成完成时（They have left.）
     - `do` → 构成疑问/否定句（Do you know?）

4. **情态动词（Modal Verbs）**

   - **定义**：表达说话者的态度（可能性、义务等）。
   - **特点**：无人称/数变化，后接动词原形。
   - 例：`can`（能力）、`must`（义务）、`should`（建议）、`may`（许可）

#### 🔄 二、按句法形式分类

1. **限定动词（Finite Verbs）**
   - 受主语人称/数/时态限制，充当谓语。
   - 例：She sings well.（第三人称单数形式）
2. **非限定动词（Non-finite Verbs）**
   - 不受主语限制，不可独立作谓语。
   - 子类：
     - **不定式（to do）**：表目的或将来（He wants to study.）
     - **动名词（doing）**：作名词性成分（Swimming is fun.）
     - **分词（doing/done）**：作定语或状语（the running boy; a broken cup）

#### 🔗 按结构组成分类

1. **单字动词（One-Word Verbs）**
   - 单一动词构成核心意义（例：run,write）
2. **短语动词（Phrasal Verbs）**
   - 动词+小品词（副词/介词），整体表意。
   - 类型：
     - **动词+副词**（及物）：`hand in`（交作业）
     - **动词+介词**（及物）：`look after`（照顾）
     - **动词+副词+介词**：`look forward to`（期待）
     - **动词+名词+介词**：`take care of`（照料）

#### 💎 特殊说明

- 兼类动词：同一动词在不同语境中可能跨类。
  - 例：sing
    - 不及物：She sings.（独立意义）
    - 及物：She sings a song.（需宾语）
- 中英差异：
  - 英语中不及物动词需加介词接宾语（`listen to music` ✓ / `listen music` ✗）

### 形容词（abjectives，abj.）

形容词（Adjectives）主要用于描述名词或代词的性质、状态、特征或属性，在句中可作定语、表语、宾语补足语等成分。

#### 📌 一、**按结构分类**

1. **简单形容词（Simple Adjectives）**
   - 由单个词构成，多为原生词或派生词。
     - **原生词**：`big`、`small`、`hot`、`cold`
     - 派生词（加前缀/后缀）：
       - 前缀：`unkind`（un- + kind）、`impossible`（im- + possible）
       - 后缀：lovely（love + -ly）、voiceless（voice + -less）
2. **复合形容词（Compound Adjectives）**
   - 由两个及以上词通过连字符组合而成，整体起形容词作用。
     - **形容词+形容词**：`bitter-sweet`（又苦又甜）
     - **名词+形容词**：`grass-green`（草绿色的）
     - **副词+分词**：`well-meant`（善意的）、`hard-working`（勤奋的）
     - 形容词+名词+-ed：kind-hearted（善良的）

------

#### 📊 二、**按句法功能分类**

1. **中心形容词（Central Adjectives）**
   - 可作定语（修饰名词）、表语（系动词后）或宾语补足语。
     - 例：green
       - 定语：`Green apples are sour.`
       - 表语：`The box is green.`
       - 宾补：They painted it green.
2. **外围形容词（Peripheral Adjectives）**
   - 仅限特定位置：
     - **仅作定语**：`utter`（`utter nonsense` ✓，`*The nonsense is utter` ✗）
     - 仅作表语：asleep（The child is asleep ✓，*an asleep child ✗）

------

#### 🔍 三、**按意义分类**

1. **品质形容词（Qualitative Adjectives）**

   - 描述性质或状态，多数有比较级和最高级：
     - `beautiful`、`honest`、`warm`
     - 例：She has a warm smile.

2. **颜色形容词（Color Adjectives）**

   - 表示颜色，无比较等级：

     - ```
       red、blue、light green（可加light/dark修饰）
       ```

3. **类属形容词（Classifying Adjectives）**

   - 表示类别、来源或材料，无程度变化：

     - ```
       Chinese（国籍）、stone（材料）、digital（类别）
       ```

4. **强调形容词（Emphasizing Adjectives）**

   - 加强语气，无比较级：

     - ```
       perfect、absolute、pure（不可说very perfect）
       ```

------

#### ⚡ 四、**按动态性分类**

1. **静态形容词（Stative Adjectives）**
   - 描述固有特征（如外观、颜色），不可与进行时搭配：
     - tall、round、blue（She is being beautiful✗）
2. **动态形容词（Dynamic Adjectives）**
   - 含动作含义，描述可改变的状态，可与进行时或祈使句搭配：
     - `careful`、`generous`、`noisy`
     - 例：Be patient!（✓），She is being rude.（✓）

------

#### 📐 五、**按等级性分类**

1. **等级形容词（Gradable Adjectives）**
   - 有程度变化，可加very修饰，有比较级/最高级：
     - `tall → taller → tallest`、`useful → more useful`
     - 例：very cold、so interesting
2. **非等级形容词（Non-gradable Adjectives）**
   - 无程度变化，无比较等级：
     - 类属词：`atomic`（原子的）、`French`（法国的）
     - 极限词：perfect、unique（不可说more perfect）

------

#### ✨ 六、**按来源分类（分词形容词）**

1. **-ing形容词（表主动/令人……）**

   - 修饰事物，说明事物对人的影响：

     - ```
       exciting（令人兴奋的）、boring（无聊的）
       ```

2. **-ed形容词（表被动/感到……）**

   - 修饰人，描述人的感受：

     - ```
       excited（感到兴奋的）、worried（担忧的）
       ```

|    类型    | 修饰对象 |           例句           |
| :--------: | :------: | :----------------------: |
| **-ing形** |   事物   | `The movie is exciting.` |
| **-ed形**  |    人    |    `We are excited.`     |

------

#### 💎 七、**其他分类补充**

- 疑问形容词（Interrogative Adjectives）

  ：which、what、whose\+ 名词（Which book?）

- 所有形容词（Possessive Adjectives）

  ：物主代词（my、your）作定语（my book）

- 数词形容词（Numeral Adjectives）

  ：基数词/序数词作定语（three apples、the second day）

------

#### ⚠️ **使用注意事项**

1. **位置规则**：

   - 定语通常前置（a red car），但修饰不定代词需后置（something important）。

   - 多个形容词修饰时，顺序为：

     大小→形状→年龄→颜色→国籍→材质（a small round yellow wooden table）。

2. **易错点**：

   - 表语形容词（如afraid、alone）不可作定语（an afraid child✗）。
   - 不可数名词需用量词修饰（a cup of water，非a water）。

3. **比较等级**：

   - 规则变化：`big → bigger → biggest`；
   - 不规则变化：good → better → best。

### 代词（pronoun，pron.）

代词是代替名词或名词短语以避免重复的词类

#### 📌一、**人称代词 (Personal Pronouns)**

代替特定人或事物，分主格（作主语）和宾格（作宾语/介词后）：

- 主格：I, you, he, she, it, we, they

  - 例句：She is a doctor.

- 宾格：me, you, him, her, it, us, them

  - 例句：The teacher praised him

  ⚠️ 常见错误：介词后必须用宾格（× between you and I→ √ between you and me）。

------

#### 📌 二、**物主代词 (Possessive Pronouns)**

表示所属关系，分两类：

- 形容词性物主代词（后接名词）：my, your, his, her, its, our, their

  - 例句：This is my book.

- 名词性物主代词（独立使用）：mine, yours, his, hers, its, ours, theirs

  - 例句：The red car is hers.

   ⚠️ 规则：名词性物主代词后不可接名词（× mine book → √ my book）。

------

#### 📌 三、**反身代词 (Reflexive Pronouns)**

表示动作作用于自身：myself, yourself, himself, herself, itself, ourselves, yourselves, themselves

- 用法：

  - 强调主语：*I myself* fixed the car. （可省略）
  - 反指动作承受者：She hurt herself. （不可省略）

  ⚠️ 限制：不可单独作主语（× Myself did it）。

------

#### 📌 四、**指示代词 (Demonstrative Pronouns)**

指代特定事物：this（这个）, that（那个）, these（这些）, those（那些）

- 例句：This is my favorite song.
- 功能：可作主语、宾语或定语（this book）。

------

#### 📌 五、**相互代词 (Reciprocal Pronouns)**

表示相互动作：each other（两者间）, one another（多者间）

- 例句：They respect each other.
- 所有格形式：each other’s ideas, one another’s homes。

------

#### 📌 六、**疑问代词 (Interrogative Pronouns)**

用于提问：who, whom, whose, what, which

- 功能：

  - *Who* called? （主语）
  - *Whom* did you meet? （宾语，正式）
  - Whose bag is this? （所属）

  ⚠️ 区别：what 泛指（What happened?），which 选择明确范围（Which color?）。

------

#### 📌 七、**关系代词 (Relative Pronouns)**

引导定语从句：who, whom, whose, that, which

- 用法：

  - 指人：The man *who* helped me.
  - 指物：The book *which* I borrowed.
  - 所属：The girl whose phone rang.

  ⚠️ 限制：非限制性从句中不可用 that（× My dog, that is brown... → √ My dog, which is brown...）。

------

#### 📌 八、**连接代词 (Conjunctive Pronouns)**

引导名词性从句：who, what, which, whoever, whatever

- 例句：What he said is true. （主语从句）。

------

#### 📌 九、**不定代词 (Indefinite Pronouns)**

不特指具体对象：

- **全体/个体**：all, each, everyone
- **部分**：some, any, none
- **否定**：nobody, nothing
- 数量：many, much, few, little

 ⚠️ 规则：

- some 用于肯定句（I need some water），any 用于疑问/否定句（Do you have any questions?）。

------

#### 📌 十、**替代词 (Substitute Pronouns)**

避免重复名词：one（单数）, ones（复数）

- 例句：I prefer the red one.

## 从句（clause）

如果担任某一成分的不是一个词或词组而是一个句子，则这个句子就是从句。



名词从句（noun clause）：和`名词`一样，在句中可担任`主语`、`宾语`（`动词宾语`和`介词宾语`）、`表语`和`同位语`，名词从句包括`主语从句`、`宾语从句`、`表语从句`和`同位语从句`。

定语从句（attributive clause）：也叫形容词性从句

状语从句（adverbial clause）

### 主语从句

由连词（that、whether）、连接代词（what、whatever、who、whoever、whom、which）和连接副词（when、where、how、why）引导，`不能省略`。



`连词`在从句中`不担任成分`，多用`it`作`形式主语`。

*`That` the earth is round* is true.

= `It` is true *`that` the earth is round*.



`连接代词`在从句中担任`主语`或`宾语`。

*`Who` let out the news* remained unknown.

= `It` remained unknown *`who` let out the news*.

*`Which` answer is right* isn't known to us.

*`Whom` she saw* didn't matter.



`连接副词`在从句中担任`状语`。

`Where` he's hiding is a secret.

*`When` the meeting will be held* has not been announced.

*`Why` he didn't come here* is not clear to anyone.

*`How` this happened* is still a question.

### 宾语从句

`if`也可以用来引导，`that`可以`省略`，可用`it`作`形式宾语`

陈述句

Mike said *`that` he was feeling ill*.



We think *`it` (is) quite right `that` all the people ought to obey the laws*.

They took `it` for granted *`that` the heavy objects went down faster than the light ones*.

一般疑问句

he asked me *`whether/if` she lived here*.

he asked me *`whether` she lived here or not*.

特殊疑问句

I wonder *`what` he likes*.

介词宾语从句

My teacher was satisfied with *`what` i did*.

某些不及物动词+介词+宾语从句，某些不及物动词后，如果跟宾语从句，则省去介词，如果跟名词作宾语，则不能省略介词

I don't care *`whether` he likes me or not*.

I don't care **about** him.

某些形容词+介词+宾语从句，在表示知觉、感情等的形容词，如sure、afraid、glad和certain等后，如果跟宾语从句，则省去介词，如果跟名词作宾语，则不能省略介词

I am sure *`that` he'll will*.

I am sure **of** his winning.

### 表语从句

表语从句是一种用来补充说明主语或宾语的从句。它通常出现在动词be、seem、appear、become等后面，用来说明主语或宾语的状态、性质、身份等，并且不能省略。

My idea is *`that` we'll all go except Lili*.

That is *`what` we should do*.

What I care most is *`whom` he loves best*.

China is no longer *`what` she used to be*.

The question is *`how` we can get there*.

### 同位语从句

如果担任同位语的是一个句子，则这个句子为同位语从句。

由`that`、`whether`、`who`、`what`引导，`不担任成分`，没有词义，`不能省略`。

从意思上来讲，同位语从句所表达的意思，就是与之同位的名词所表达的意思。

与that从句同位的名词必须是一个表示事实等的`抽象名称`，常用的有：

news, idea, belief, doubt, fact, rumour, question, order, hope, thought, reply, answer



The rumour *`that` there'll be the earthquake* soom spead.

He didn't answer the question *`whom` he got help from*.

### 定语从句（Attributive clause）

先行词

Mother is **the person** *`whom` he misses most*.

关系词

关系副词，where、when、how和why等起着`副词或介词短语`和`连词`的作用

This is **the place** *where i lost my wallet*

## 动词不定式（infinitive）

定义：`动词不定式`与`动名词`、`分词`一样，都是动词的一种`非谓语形式`。

形式：`to` + 动词原形，这里的`to`不是`介词`，一般叫它小品词，它只是一个符号，没有词性，有时可以省略。

用法：除了不能作谓语外，可以担当其他任何成分：`主语`、`宾语`、`表语`、`状语`、`定语`和`宾语补足语`，在句子中起`名词`、`形容词`和`副词`的作用。

### 不定式作名词

#### 不定式作主语

`To abey the laws` is important.

不定式做主语时，往往用it做形式主语

It is enough to have one close friend in one's life.

#### 不定式作表语

My wish is `to be a doctor`.

#### 不定式作宾语

I want `to do it myself`.

当不定式在consider、find、make、regard、think和feel等动词后作宾语时，一般用it作形式宾语代理不定式，同时把不定式置于补语之后，即主语 + 动词 + it + 补语 + to do结构

We think it quite important for us to learn a foreign language well.

### 不定式作形容词

动宾关系

He always has a lot of meetings `to attend`.

主谓关系

The factory `to produce electricity` will be set up next year.

to + 不及物动词 + 介词：被修饰的词成为这个介词的逻辑宾语

He is looking for a room `to live in`.

名词或代词（地点和工具等） + to + 动作性动词 + 介词

Please pass me some paper `to write on`.

Please lend me somthing `to write with`.

当被修饰的词是place、time和way时，可以省略介词

He has no money and no place `to live (in)`.

I think the best way `to travel (by)` is on foot.

There is no time `to think (about)`.

### 不定式作宾语补足语

We allow you `to enter the hall`.

### 不定式作副词

表原因

He is lucky `to get here on time`.

表目的

He came `to help me with my maths`.

表结果

I hurried to get there only `to find him out`.

After that day they were seperated. Never to see each other again.

## 动名词

动名词由`动词原形`+`ing`构成，跟现在`分词形式`一样，动名词在句子中起`名词`的作用，可以作`主语`、`表语`、`宾语`（介词宾语和动词宾语）和`定语`。

动名词作主语

`Learning` English is difficult.

动名词作表语

His job is `painting`.

动名词作宾语

Our monitor suggested `having` a discussion of this subject.

动名词作定语

动名词一般只以·`单个的形式`而不能以短语的形式做`定语`，它的位置只能放在它所修饰词的`前面`。动名词作定语一般表示所修饰词的`用途`。

a `sleeping` car

a `swimming` pool

动名词的否定式

not + 动名词

I am sorry for `not being` here.

动名词短语

物主代词或名词的所有格 + 动名词，物主代词或名词的所有格可以理解为被修饰词的`逻辑主语`，动名词短语可以作主语、表语和宾语。

Do you mind `my leaving` now?

`Li Ming's being` late made his teacher very angry.

## 现在分词和过去分词

分词具有`形容词`和`副词`属性质，可以作`表语`、`定语`、`状语`和`宾语补足语`。

分词作表语

The story is `interesting`.

I am `interested` in it.

分词作定语

This is a `moving` film.

分词作状语

The secretary worked late into the night, `preparing` a long speech for the president.

`Given` more attention, the trees could have grown better.

分词作宾语补足语

When he passed the bank, he saw the thief `stealing` some money from the bank.

When he was decorating his house, he got the bedroom `painted` first.

## 直接引语和间接引语

She said: "I like English very much".

She said that she liked English very much.

### 1.一般现在时（simple present）

**（1） 表示现在经常发生的动作或存在的状态。**

She plays the violin after school every afternoon.

她每天下午放学后拉小提琴。

**（2）表示现在的特征、状态、能力、性格等。**

This job calls for great patience.这份工作需要极大的耐心。

**（3）表示客观事实、普遍真理或名言警句等。**

Light travels faster than sound.光比声音传播得快。

**（4）一般现在时表将来**

a. 在`时间/条件/让步状语从句`中，常用一般现在时表将来。（即：主将从现）

Even if it rains tomorrow，the sports meeting will take place.即使明天下雨，运动会也照常举行。

b. 表示按时间表拟定的或安排好的到时候就发生的事情或要发生的动作。

The plane arrives at 7:00 pm.There is only a little time left.火车下午七点到达，我们只有一点儿时间了。

**（5）表示正在发生的动作或存在的状态。**

常用在以here，there等开头的倒装句中。There goes the bell . 铃声响了。

### 2. 一般过去时（simple past tense）

**（1） 表示过去某个时间发生的事，存在的状态或过去反复发生的动作。**

we both traveled around Europe for about half a month last summer.

去年夏天我们两个人去欧洲大概旅行了半个月。

**（2）表示过去经常性或习惯性的动作。**

when I was in primary school，I often swim in the river.

当我上小学时，我常常在河里游泳。

### 3.一般将来时（the simple future tense）

（1）will do

**a. 表示主观意愿的将来。**

I will visit my grandparents this weekend.

这周末我将会去看望我祖父母。

**b. 表示客观将来。**

Maybe people will live up to be over 100 years old in 100 years.

一百年后也许人们会活到100多岁。

**c. 表示临时决定。**

A: Lucy has been ill for 3 days。

B: oh, I'm sorry to hear that，I will go and see her.

（2）am/is/are going to do

**a. 表示有迹象表明要发生的动作。**

It looks as if it is going to rain.

看来天要下雨了。

**b. 表示现在打算、计划、准备要做某事。**

They are going to take a picnic next week.

他们打算下周去野餐。

（3）am/is/are to do

**a. 表示注定要发生的事情。**

His plan is to be a failure.

他的计划注定要失败。

**b. 表示说话人的职责、意志、意图、义务、命令等。**

All students are to hand in your homework within three days.

所有的学生三天之内必须递交你们的家庭作业。

**c.表示预先安排好的计划或约定。**

The president is to visit China next month.

总统下月将访华。

（4）am/is/are doing

常用于这种结构的动词有：come，go，start，begin，leave，run，stay，do，take等。

He’s leaving early tomorrow morning.

明天早上他很早就出发。

我以为你已经走了。

### 16. 过去将来完成进行时（the past future perfect continuous tense）

**表示就过去某一时间而言 ，将来某一时刻之前一直在进行的动作，是否持续下去，根据上下文而定。**

I would have been waiting for you before 10 years.

从十年前开始，我都一直在等你。

## 被动语态

## 助动词（auxiliary verb）

`助动词`本身`没有词义`，主要帮助构成`谓语`，表示`时态`、`语态`、`语气`或构成`否定`或`疑问`形式，助动词在句子中必须同`主语`的`人称`和`数`一致。

常用的五个助动词`be`、`do`、`have`、`shall`和`will`

| 原型 | 一般现在时  | 过去时    | 过去分词 |
| ---- | ----------- | --------- | -------- |
| be   | Is, am, are | was, were | been     |
| do   | do, does    | did       | done     |
| have | have, has   | had       | had      |
| -    | shall       | should    | -        |
| -    | will        | would     | -        |

## 动词语气（verb‘s mood）

### 陈述语气

对情况的叙述或询问，使用`陈述语气`，用于`陈述句`、`疑问句`和`感叹句`

### 祈使语气

提出劝告、请求、命令等时，使用`祈使语气`，用于`祈使句`。

两种句型：

1. 句型一：

肯定句：`动词原形`，Be careful!

否定句：`Don't 动词原形`，Don't run!

2. 句型二：

肯定句：`Let + 宾语 + 动词原形`，Let's go.

否定式：`Let + 宾语 + not + 动词原形`，Let‘s not talk about it now.

### 虚拟语气

#### 虚拟语气用于条件句

##### 非真实条件句

如果`假设`的情况`完全不存在`或者实现的`可能性很小`，`几乎没有`时，就需要用`虚拟语气`来表示，这时的条件句是`非真实的条件句`，此时的句子时态比真实条件句中的时态后退一步，即：

现在时-》过去时

过去时-》过去完成时

将来时-》过去将来时

过去将来时-》过去将来完成时

非真实条件句（以do和be为例）

与`现在`事实相反

| 从句                               | 主句                                |
| ---------------------------------- | ----------------------------------- |
| 用`过去时`                         | 用`过去将来时`                      |
| if + 主语 + `were/did/were doing`, | 主语 + should/would/might/could do. |
| if i `were` you,                   | i `should/would` go at once.        |
| If there `was` no air,             | people `would` die.                 |

与`过去`事实相反

| 从句                                | 主句                             |
| ----------------------------------- | -------------------------------- |
| 用`过去完成时`                      | 用`过去将来完成时`               |
| If + 主语 + `had done`,             | 主语 + `should/would have done`. |
| If you `had been` there last night, | nothing `would have happened`.   |
| If you `had got` there earlier,     | you `would have caught` the bus. |

与`将来`事实相反

| 从句                                        | 主句                                       |
| ------------------------------------------- | ------------------------------------------ |
| 用`过去时`（were to）：should + V（万一）   | 用`过去将来时`                             |
| If + 主语 + `were/did/should/were to do`,   | 主语 + `should/would/might/could` do.      |
| If it `were` Sunday tomorrow,               | we `wouldn't have` an exam.                |
| If you `were to` visit the school tomorrow, | you `would see` me.                        |
| If you `should` meet him, 万一你见到他      | you `would say` "hello" to him. 你向他问好 |

注意：

虚拟语气的were除了在if i were you的结构中不能改动外，其他情况有时可用was。
