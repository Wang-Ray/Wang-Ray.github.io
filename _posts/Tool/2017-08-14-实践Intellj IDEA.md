### 快捷键

| 快捷键         | 功能                   |
| -------------- | ---------------------- |
| Ctrl+I         | 自助接口实现选择和生成 |
| Ctrl+O         | 自助Override选择和生成 |
| Ctrl+Shift+L   | 格式化                 |
| Ctrl+Q         | 查看注释               |
| F8             | step over              |
| F7             | step into              |
| alt+shift+F7   | force step into        |
| shift+F8       | step out               |
| F9             | resume program         |
| alt+F9         | run to cursor          |
|                |                        |
| shift+F6       | rename                 |
| alt+enter      | import                 |
| alt+insert     | generate               |
|                |                        |
| ctrl+alt+<-    | 返回到上一个光标点     |
| ctrl+alt+->    | 前进到下一个光标点     |
|                |                        |
| ctrl+shift+F10 | run                    |
